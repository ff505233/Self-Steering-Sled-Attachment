# Self-Steering-Sled-Attachment

iPhone bluetooth controlled rear control system for steering sleds in the snow

Hardware: Arduino Uno R3, gobuilda 64kg/cm planetary gear motor, Snappy XO V1.0 motor driver, HM-10 Bluetooth module

Libraries Required: SoftwareSerial(Built in)  & MicroBlue(Files included above):serial communication with bluetooth module.
